# Demo Script

Hello, my name is [Your Name], and this is my ACC102 GitHub data analysis mini project.

The project is called **Python-Based Stock Market Analysis of Major Technology Companies**. I used Python to analyse five well-known technology stocks: Apple, Microsoft, NVIDIA, Alphabet, and Amazon.

For the data, I used five years of daily historical stock prices downloaded from Yahoo Finance through the `yfinance` library. I focused mainly on adjusted closing prices, because they are more suitable for comparing historical performance over time.

In the analysis, I cleaned the downloaded data and then calculated several simple financial measures. These included daily returns, cumulative returns, rolling 30-day volatility, 20-day and 50-day moving averages, the correlation between stock returns, and maximum drawdown. I also created charts and a summary table to make the comparisons easier to understand.

The main purpose of the project was not to predict future prices, but to show how basic Python tools can help us understand stock behaviour more clearly. The results help highlight differences in long-term performance, differences in risk, and the fact that large technology stocks often move together.

In this repository, I included the main Python script, a Jupyter notebook version of the analysis, the generated figures and tables, a README, a reflection draft, and this demo script.

Overall, this project shows a complete but simple workflow for business data analysis using public financial data. Thank you for watching.
